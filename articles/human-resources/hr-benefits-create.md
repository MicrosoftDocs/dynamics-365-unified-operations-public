# Create a new benefit



[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

This task shows you how to create benefit elements that you use when creating a new benefit. The demo data company used to create this task is USMF. This task is intended for a Compensation and Benefits manager.


## Create benefit elements

1. Go to **Human resources > Benefits > Setup > Benefit elements**.
1. Select **New**.
1. In the **Type** field, enter the name of the type of benefit that you're creating.
1. In the **Description** field, enter a value.
1. In the **Concurrent enrollment** field, select an option.

    To restrict employees' ability to enroll in multiple medical plans, select **One enrollment per type**.

1. In the **Payroll category** field, select an option.
1. On the **Plans** tab, select **New**.
1. In the **Plan** field, enter a value.
1. In the **Description** field, enter a value.
1. In the **Type** field, enter or select a value.
1. In the **Payroll impact** field, select an option.
1. Select **Save**.

## Create a benefit

1. Go to **Human resources > Benefits > Benefits**.
1. Select **New**.
1. In the drop-down dialog box, enter or select a value in the **Plan** field.
1. Enter or select a value in the **Option** field.
1. Enter a date and time in the **Effective** field.
1. Select **Create benefit**.
