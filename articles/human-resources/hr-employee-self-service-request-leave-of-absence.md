# Request a leave of absence

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

In Dynamics 365 Human Resources, you can submit a request for a leave of absence and check the status of your leave requests.

## Request a leave of absence

1. In the **Employee self service** workspace, select **More** (...) in the **Time Off Balances** tile.
2. To submit a leave of absence request, select **Request leave of absence**.
1. Enter information for **Leave type**, **Start date**, and **End date**.
1. If you need to submit any supporting documentation, select **Upload** under **Attachments**.
1. Enter information in **Comment**, if needed.
1. Select **Submit** when you're ready to submit your request. Otherwise, select **Save draft**.

## View leave of absence request status

1. In the **Employee self service** workspace, select **More** (...) in the **Time Off Balances** tile.

2. To view your leave of absence requests, select **View leave of absence**.

## Update a leave of absence request

1. In the **Employee self-service** workspace, on the **Leave of absence** tile, select **More (...)**.
1. Select the leave of absence request to update, and then select **Update leave of absence**.
1. In the **End date** field, update the value as appropriate to extend or shorten the leave of absence.
1. If you confirm the end date, set the **Confirm end date** option to **Yes**.
5. After the **Confirm end date** option is set to **Yes**, you can upload a return-to-work notice. Then select the checkbox to confirm that a return-to-work notice has been uploaded.
1. Select **Submit** to update the leave of absence request.

## Cancel a leave of absence request

1. In the **Employee self-service** workspace, on the **Leave of absence** tile, select **More (...)**.
1. Select the leave of absence request to cancel, and then select **Update leave of absence**.
1. Set the **Cancel leave** option to **Yes**.
1. Select **Submit** to cancel the leave of absence request.

## Import leave requests from other systems or older systems

To import leave requests from another system, go through the regular workflow to create the appropriate leave transactions. Alternatively, you can import the leave bank transactions and the leave requests in a completed state. If you import only the leave requests, the leave bank transactions aren't automatically created.
