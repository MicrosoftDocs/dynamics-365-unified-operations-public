# Set up injury and illness information

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Employers need to know when their employees suffer injuries or illnesses because of hazards in the workplace. Use the **Injury and illness** page to set up information that facilitates reporting of workplace injuries or illnesses. You can set up types of injuries and illnesses, including types of treatments, costs, and outcomes. This procedure uses the **USMF** demo data company.

1. Go to **Human resources** > **Workers** > **Injury and illness** > **Injury and illness setup**.
1. Select **New**.
1. In the **Injury or illness type** field, enter a value (for example, **Fracture**).
1. In the **Description** field, enter a value (for example, **Bone fracture**).
1. Select the **Body parts** tab.
1. Select **New**.
1. In the **Body part** field, enter a value (for example, **Wrist**).
1. In the **Description** field, enter a value (for example, **Wrist**).
1. Select the **Treatment types** tab.
1. Select **New**.
1. In the **Treatment type** field, enter a value (for example, **Splint**).
1. In the **Description** field, enter a value (for example, **Put a splint on**).
1. Select the **Cost types** tab.
1. Select **New**.
1. In the **Cost type** field, enter a value (for example, **X-rays**).
1. In the **Description** field, enter a value (for example, **X-rays**).
1. Select the **Outcome types** tab.
1. Select **New**.
1. In the **Outcome type** field, enter a value (for example, **Therapy**).
1. In the **Description** field, enter a value (for example, **Physical therapy**).
1. Select the **Severity levels** tab.

    Create customizable severity levels. For example, severity 1 might indicate a minor injury, whereas severity 3 might indicate a severe injury.

1. Select the **Reporting agencies** tab.

    Reporting agencies are the agencies that the incident must be reported to. Select the **Default** checkbox for the agency that is the default agency that injuries and illnesses must be reported to.

1. Select **Save**.
