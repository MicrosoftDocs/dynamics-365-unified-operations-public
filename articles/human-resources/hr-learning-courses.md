# Create a course

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Human resources administrators and managers can use the courses features to maintain information about the training that they offer to workers.

## Course templates

Use course templates to categorize courses according to their structure or content. Use them to prefill information when you create a new course. Create course templates on the **Course templates** page.

The minimum and maximum number of participants that you can register for a course is defined on the **General** FastTab of the **Courses** page.

## Course setup type

The following table lists the three setup types for courses. Setup types determine the structure of the course.

| Setup | Description |
|---|---|
| **Standard** | Select this type for courses that don't have a daily agenda. This setup type is the default when you create a new course. |
| **Agenda** | Select this type to plan the details of each day of a course that takes place over multiple days. |
| **Agenda + session** | Select this type for the more complex courses. For example, you can divide the agenda for the course into tracks and sessions.<ul><li>**Track** – Tracks are specific subject areas for a course.</li><li>**Sessions** – Sessions divide up tracks and help identify specific processes or techniques that are relevant to the track.</li></ul> |

## Course status

The **Course status** field is at the top of the page. The following table lists the possible course statuses and the actions you can take when the course has a specific status.

| Status | Actions |
|---|---|
| **Draft** | <ul><li>Create and modify course information.</li><li>Change the course status to **Open** so that workers can register for the course. (In-person only)</li></ul> |
| **Open** | <ul><li>Register participants for the course.</li><li>Remove participants from the course.</li><li>Confirm participants for the course.</li><li>Change the course status to **Closed** or **Canceled**.</li><li>Plan questionnaires for participants whose status is **Confirmed**.</li></ul> |
| **Closed** | You can reopen the course. |
| **Canceled** | You can reopen the course. |
