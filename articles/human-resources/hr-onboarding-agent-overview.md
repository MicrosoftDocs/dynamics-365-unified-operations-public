# Dynamics 365 Human Resources Onboarding agent (preview)

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]
[!INCLUDE [preview-banner](~/../shared-content/shared/preview-includes/preview-banner.md)]

The Dynamics 365 Human Resources Onboarding agent (preview) is a guided experience that coordinates Human Resources professionals and new hires through onboarding as a single, connected journey in Microsoft Teams. It automates common onboarding setup tasks, recommends the next action, and writes every confirmed action back to Dynamics 365 Human Resources, so records stay consistent and ready for downstream payroll processing. Compensation details flow to Finance, which helps ensure that new hires are ready for their first pay cycle.

The onboarding agent provides two experiences that work together:

- Onboarding agent for HR
- Onboarding agent for new hires

## Key capabilities

- Works in the flow of work – Both the HR manager and new hire experiences run in Microsoft Teams, so users don't switch between systems.
- Grounded in your system of record – Every action the agent takes writes back to Dynamics 365 Human Resources.
- Intelligent data capture – The agent uses recruitment records, uploaded documents, and natural-language input to draft records, recommend checklists, leave plans, and extract key details. Users review and confirm the information instead of entering it manually.
- Multi-role by design – Two tailored experiences, one for HR managers and one for new hires, work together so hand-offs are a continuous flow.

## Onboarding agent for HR

The Onboarding agent for HR helps Human Resources professionals complete onboarding activities in the flow of work in Teams:

- **Onboarding checklist recommendations** – Suggestions in Teams that help Human Resources professionals assign the most relevant onboarding tasks during worker creation.
- **Leave plan recommendations** – Streamlined recommendations for assigning appropriate leave plans directly in the flow of work.
- **Compensation details** – The ability to enter compensation details for candidates while you create worker records.
- **Onboarding summary** – A consolidated view in Teams that summarizes the onboarding process. This view helps Human Resources professionals track progress and take necessary actions.

## Onboarding agent for new hires

The Onboarding agent for new hires guides new employees through the steps to complete their onboarding in Teams:

- New hires receive a welcome kit and a list of learning materials that they can review as they join the organization.
- New hires validate their personal details and add their address and contact details.
- New hires upload identification documents.
- New hires send the welcome email.
