# Manage assortments

[!include [banner](../includes/banner.md)]

This article explains how to create and publish a new product assortment in Microsoft Dynamics 365 Commerce.

The following procedure demonstrates how to create and publish a new product assortment and uses the demo data company USRT.

To create and publish an assortment, follow these steps.

1. In Commerce headquarters, go to **Category and product management**.
1. Select the **Assortments** tab.
1. Select **New**.
1. Select **Assortment**. You need to provide an **Assortment ID** and it must be a unique value.  
1. In the **Assortment name** field, enter a value.
1. In the **Effective date** field, enter a date.
1. In the **Expiration date** field, enter a date.
1. Expand the **Commerce channels** section.
1. Select **Add line**.
1. In the tree, select **Contoso Retail\Electronics\Boston**.
1. Select **Add**.
1. Select **OK**.
1. Expand the **Products** section.
1. Select **Add line**.
1. In the **Category** field, enter or select a value.
1. Select **Save**.
1. Select **Publish**.
1. Select **Yes**.
