# Get support for finance and operations apps or Lifecycle Services (LCS)

[!include [banner](../../../finance/includes/banner.md)]

This article explains how to get help with finance and operations apps or Microsoft Dynamics Lifecycle Services (LCS). 

| Task | More information |
|---|---|
| Ask the community. | Go to the [Dynamics 365 Community](https://community.dynamics.com/) page to get help with your questions from the Microsoft Dynamics community. |
| Get help with questions about licensing. | Contact your partner or a Microsoft sales representative. |
| Use the **Issue search** tool. | In [Lifecycle Services](https://lcs.dynamics.com/), use the **Issue search** tool to quickly search for Microsoft Knowledge Base (KB) articles, hotfixes, and workarounds for reported problems. You can see which reported problems are in the process of being fixed for a specific functional area, and which problems are already fixed. For more information, see [Issue search Lifecycle Services](../../dev-itpro/lifecycle-services/issue-search-lcs.md). |
| Get support within your finance and operations app. | Select the **Help** button (**?**) in the upper-right corner of the app, and then select **Support**. Report problems on the **Active problems** tab in LCS. There, admins can determine whether they should provide in-house support or submit the problems to Microsoft. |
| Open a support ticket with the Microsoft Support team. | In [Lifecycle Services](https://lcs.dynamics.com/), the **Support** tile opens a tool that helps you manage support incidents. To submit problems directly to Microsoft, select the **Support** tile in your LCS project. You can then submit problems in two ways: On the **Active problem** tab, select your problem, and then select **Submit to Microsoft**. On the **Submitted to Microsoft** tab, select **Submit an incident**, and then follow the on-screen instructions to submit the incident. After you submit an incident, you receive an email message from the Microsoft Support engineer who is assigned to your case. |
| Request new features and functionality. | Visit [Dynamics 365 Application Ideas](https://experience.dynamics.com/ideas/) to view, search, or vote for existing ideas, or to add new ideas. |
