# Develop and open job requisition

[!include [banner](../../includes/banner.md)]

> [!IMPORTANT]
> The recruiting functionality in this article will be referred to as Recruitment projects and focuses on applicants, applications, and recruitment projects.  

Recruitment projects help you manage the recruiting process. For each recruitment project, you can set up information such as the job that recruiting is done for, the name of the recruiter, the status of the project, and the department where the job is located. After you create a recruitment project, you can write a job advertisement for it, publish the ad on the **Employee self service** page, associate applications for employment with the project, and track activities for that project. This procedure uses the **USMF** demo data company.

1. Go to **Human resources > Recruitment > Recruitment projects > Recruitment projects**.
1. Select **New**.
1. In the **Recruitment project** field, enter a value.
1. In the **Description** field, enter a value.
1. In the **Recruiter** field, select the drop-down arrow to open the lookup.
1. In the list, find and select the desired record.
1. In the list, select the link in the selected row.
1. Select **Select**.
1. In the **Department** field, select the drop-down arrow to open the lookup.
1. In the list, select the link in the selected row.
1. In the **Job** field, select the drop-down arrow to open the lookup.
1. In the list, find and select the desired record.
1. In the list, select the link in the selected row.
1. In the **Number of openings** field, enter a number.
1. In the **Hiring manager** field, select the drop-down arrow to open the lookup.
1. In the list, find and select the desired record.
1. In the list, select the link in the selected row.
1. Select **Select**.
1. In the **Application deadline** field, enter a date.
1. Select **Media**.

    Recruitment projects include the option to specify media outlets that you use to advertise open positions.

1. Select **New**.
1. In the **Media** field, select the drop-down arrow to open the lookup.
1. In the list, select the link in the selected row.
1. In the **Start date** field, enter a date.
1. In the **End date** field, enter a date.
1. Select **Save**.
1. Close the page.
1. Select **Job ads**.
1. Select **Save**.
1. Close the page.
1. Select or clear the **Display on employee self service** checkbox.

    Select the checkbox to make the recruitment project visible to employees on their **Employee self service** page.

1. Select the **Recruitment project** status.
1. Select **Start**.

    The **Started** status indicates that the project is ready to receive applications.

1. Select **OK**.
