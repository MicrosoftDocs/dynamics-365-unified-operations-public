# Configure the global address book

[!include [banner](../../includes/banner.md)]

Use this procedure to set the default values and security policies for the global address book.

The demo data company used to create this task is USMF. This task is intended for the Planning and configuration team.

1. In the navigation pane, go to **Modules** > **Organization administration** > **Global address book** > **Global address book parameters**.
1. In the **Name sequence** field, select how names should be shown.
1. In **Delete parties with no roles**, select whether to delete parties that don't have an assigned role.
1. In **Use duplicate check**, select whether to check for duplicate records.
1. In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.
1. In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.
1. In the **Party** field, select an option.
1. In the **Customer** field, select an option.
1. In the **Vendor** field, select an option.
1. In the **Prospect** field, select an option.
1. In the **Competitor** field, select an option.
1. Select the **Private location security** tab.
1. In the list, find and select the desired record. Press the **Shift** key to select multiple roles to add to the **Selected roles** pane, and then select the arrow to add the selected roles.  
1. Select **Save**.
