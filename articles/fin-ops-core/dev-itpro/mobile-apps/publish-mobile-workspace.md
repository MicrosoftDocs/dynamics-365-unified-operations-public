# Publish mobile workspaces

[!include [banner](../includes/banner.md)]
[!include [mobile app deprecation](../includes/mobile-app-deprecation-banner.md)]

This article describes the steps that system administrators must follow to publish a mobile workspace. You must publish a mobile workspace so that users can access it in the mobile app. 

## Publish a mobile workspace

To publish a mobile workspace, follow these steps:

1. In your browser, start your web client.
1. Select **Settings** > **Mobile app**.
1. Select the mobile workspace to publish.
1. Select **Publish**.

After you publish a new workspace, users must pull to refresh the list of mobile workspaces.
