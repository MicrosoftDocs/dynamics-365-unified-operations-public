# Respond to privacy data requests resources

[!include [banner](../includes/banner.md)]

This article provides links to information that can help you respond to a request for private information as a customer using Dynamics 365 Finance, Supply Chain Management, Commerce, Human Resources, and Microsoft Dynamics AX 2012.

Your first step in responding to a request for data is usually to use the Person search report to locate the requested data. In some cases, you might need to use other reports, access specific pages in the product that you're using, or extend the Person search report. (The report isn't currently available for Microsoft Dynamics AX 2012.) This article points to content that helps you complete those tasks.

## Compliance Manager

Compliance Manager is a cross–Microsoft Cloud services solution that's designed to help organizations meet complex compliance obligations. It performs a real-time risk assessment that reflects your compliance posture against data protection regulations when using Microsoft Cloud services, along with recommended actions and step-by-step guidance.

### Compliance Manager

Try Compliance Manager yourself by visiting [https://aka.ms/compliancemanager](https://aka.ms/compliancemanager).
