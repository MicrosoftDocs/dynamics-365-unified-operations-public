# LCS project creation freeze

[!include [banner](../includes/banner.md)]

As of February 16, 2026, new cloud implementation projects can no longer be created in Microsoft Dynamics Lifecycle Services (LCS) for Dynamics 365 Finance, Supply Chain Management, and Project Operations. New customers for these products are directed to the [Power Platform admin center](/power-platform/admin/unified-experience/finance-operations-apps-overview) for environment and project management.

## What changed

New LCS project creation is frozen for new Dynamics 365 Finance, Supply Chain Management, and Project Operations customers. This change is part of the broader transition to the [Power Platform admin center (PPAC)](/power-platform/admin/unified-experience/finance-operations-apps-overview) as the unified administration experience for these products.

## When did this take effect

The freeze on new LCS project creation became effective on February 16, 2026. A code freeze was implemented in January 2026 to prepare for this change.

## Who is affected

This change affects new customers and tenants who are starting new cloud implementation projects for the following products:

- Dynamics 365 Finance
- Dynamics 365 Supply Chain Management
- Dynamics 365 Project Operations

## Who is NOT affected

The following scenarios are not affected by this change:

- **Existing customers** who already have active LCS projects can continue to use them.
- **Dynamics 365 Commerce** projects continue to be created in LCS.
- **AX 2012 upgrade** projects are unaffected.
- **On-premises implementations** can still create LCS projects as needed.

## What to do instead

New customers should use the [Power Platform admin center](https://aka.ms/OneAdmin-Overview) to manage their Dynamics 365 Finance, Supply Chain Management, and Project Operations environments. The Power Platform admin center provides a unified administration experience for these products.

## Exception process

If you have a business-critical need to create a new LCS project, you can submit an exception request through the [LCS Project Creation Exception form](https://aka.ms/LCSProjectCreationException). Exception requests are reviewed on a case-by-case basis. Approval is not guaranteed and depends on the specific circumstances of the request.
