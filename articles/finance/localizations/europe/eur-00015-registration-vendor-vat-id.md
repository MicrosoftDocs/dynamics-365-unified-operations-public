# EUR-00015 Registration of vendor VAT ID

[!include [banner](../../includes/banner.md)]

This article explains how to add VAT registration IDs and a tax exempt number to a vendor account. This process is similar for legal entities and customers in all European countries and regions.

Before you complete this procedure, set up VAT IDs.

1. Go to **Accounts payable** > **Vendors** > **All vendors**.
1. In the list, find and select the vendor record you want to work with.
1. Select **Registration IDs** > **Add**.
1. Select **VAT ID** and in the **Registration number** field, enter a value. The ID must match the specified format of the registration type.  
1. On the **General** tab, enter a date in the **Effective** field and then select **Save**.
1. Select **New** and enter values in the **Name** and **Description** fields.
1. In the **Country/region** field, enter or select a value.
1. In the **Primary for country/region** field, select **Yes** and then select **Save**.
1. On the **Overview** tab, select **Add**.
1. In the **Registration type** field, enter or select a value.
1. In the **Registration number** field, enter a value. For example, specify a VAT ID. The ID must have the same format as the registration type.  
1. Select **Save** and close the page.
1. In the list, find and select the record you want to update.
1. In the list, in the selected row, select the link.
1. Expand the **Invoice and delivery** section and select **Edit**.
1. In the **Tax exempt number** field, enter or select a value and then select **Save**.
