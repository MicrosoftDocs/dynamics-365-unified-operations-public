# Set up asset group depreciation for depreciation books (India)

[!include [banner](../../includes/banner.md)]

Use asset group depreciation for assets that are similar and for assets that have the same number of years of life. The asset group depreciation method is different from other methods of depreciation where you depreciate assets individually or group dissimilar assets together. The demo data company used to create this procedure is INMF.

1. Go to **Fixed assets** > **Setup** > **Books**.
1. Select **New**.
1. In the **Book** field, enter a value.
1. In the **Description** field, enter a value.
1. In the **Depreciation profile** field, enter or select a value.
1. Select **Yes** in the **Calculate depreciation** field.
1. In the **Posting layer** field, select an option.
1. In the **Posting layer** field, select an option.
1. In the **Posting layer** field, select an option.
1. Select **Yes** in the **Allow negative net book value** field.
1. Select **No** in the **Allow negative net book value** field.
1. Select **Yes** in the **Allow net book value higher than acquisition costs** field.
1. Select **No** in the **Allow net book value higher than acquisition costs** field.
1. In the **Calendar** field, enter or select a value.
1. Select **Fixed asset groups**.
1. In the **Fixed asset group** field, enter or select a value.
1. Select **Save**.
