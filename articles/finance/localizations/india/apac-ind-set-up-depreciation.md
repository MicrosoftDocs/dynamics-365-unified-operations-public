# Set up parameters to calculate depreciation (India)

[!include [banner](../../includes/banner.md)]

This procedure shows you how to set up fixed assets parameters to calculate depreciation based on the Companies Act of India and generate the Depreciation schedule report. The demo data company used to create this procedure is INMF.

1. Go to Fixed assets > Setup > Fixed assets parameters.
1. Expand the India depreciation section.
1. Select **Yes** in the Companies Act depreciation field.
1. In the Min. working days for seasonal industries field, enter a number.
1. In the Min. working days for non-seasonal industries field, enter a number.
1. In the Max. acquisition value to avail full depreciation field, enter a number.
1. Select **Save**.
