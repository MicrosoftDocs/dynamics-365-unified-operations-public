The following Copilot experiences are available in Microsoft Sustainability Manager.

| Feature | Description | Learn more |
|---------|-------------|----------------|
| Analyze documents with Copilot (preview) | Upload ESG-related documents and ask Copilot questions about the content. | [Analyze documents with Copilot (preview)](/industry/sustainability/sustainability-manager-copilot-document-analysis) |
| Create calculation models with Copilot (preview) | Describe the calculation model that you want to create, and Copilot generates the model for you. To use this feature, turn it on in **Application settings**, and then open it from the **Models** page. | [Create calculation models with Copilot](/industry/sustainability/sustainability-manager-copilot-calculation-models) |
| Generate reports with Copilot (preview) | Generate an emissions report or a Corporate Sustainability Reporting Directive (CSRD) report. Start report generation from the Copilot side pane. | [Generate preparation reports with Copilot](/industry/sustainability/sustainability-manager-copilot-reports) |
| Query data with Copilot (preview)| Ask Copilot questions about your organization's data to quickly find values such as an assessment report metric or an organizational unit's revenue. Start data querying from the Copilot side pane. | [Query data with Copilot](/industry/sustainability/sustainability-manager-copilot-data-query) |
| Find facts with Copilot (preview) | Draft qualitative and quantitative responses for ESG disclosures. Copilot finds relevant data in your uploaded documents and suggests facts that meet disclosure requirements. Review and edit the facts before you add them to reports. | [Find facts with Copilot](/industry/sustainability/sustainability-manager-copilot-find-facts) |
